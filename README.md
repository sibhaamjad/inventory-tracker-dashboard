# Inventory Tracker Dashboard

An Excel-based inventory management system designed to monitor stock levels, track item movement across multiple suppliers, and surface low-stock alerts through an automated summary dashboard.

## Overview
The workbook maintains a centralized inventory log of 100+ items across five categories (Electronics, Furniture, Networking, Accessories, Stationery) and converts raw stock data into actionable insights via a dedicated dashboard view.

## Features
- **Inventory Log**: Tracks Item ID, Name, Category, SKU, Supplier, Unit Cost, Quantity in Stock, Reorder Level, Storage Location, and Last Restocked date
- **Automated Status Flagging**: Items are automatically flagged as "OK" or "Low Stock" based on reorder level thresholds
- **Summary Dashboard**: Live KPIs including Total Items, Total Stock Value, Low Stock Item Count, and Total Categories
- **Category Breakdown**: Aggregated stock totals by category for quick distribution analysis
- **Multi-Supplier Tracking**: Monitors inventory across multiple suppliers (StorageMax, TechSource, CableWorld, NetLink, OfficeMart, PaperHub)

## Tools Used
- Microsoft Excel (formulas, conditional logic, dashboard summary view)
